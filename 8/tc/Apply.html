Apply

Equivalent to Applicative without the pure method.

A strong lax semi-monoidal endofunctor.

The “semi-“ prefix usually means “without identity”; the identity for an Applicative (a strong lax monoidal endofunctor) is pure.

Apply relates to Semigroup as Applicative relates to Monoid.

This statement can be made exact by recognizing the category in which Applicative is a Monoid; Apply is a semigroup in that category.

import scalaz.Scalaz._

Law

The only law introduced in addition to the Functor laws is:

scala>   def applyAssoc[F[_], A, B, C, T](in: F[A])(fst: F[A => B],
     |                                              snd: F[B => C])(assert: (F[C], F[C]) => T)(implicit F: Apply[F]): T = {
     |     import F.{ ap, map }
     |     def compose(f: B => C)(g: A => B): A => C = f compose g
     |     assert(
     |       ap(ap(in)(fst))(snd),
     |       ap(in)(ap(fst)(map(snd)(compose)))
     |     )
     |   }
applyAssoc: [F[_], A, B, C, T](in: F[A])(fst: F[A => B], snd: F[B => C])(assert: (F[C], F[C]) => T)(implicit F: scalaz.tc.Apply[F])T

That is to say, ap is associative composition of static arrows in F[_], where type Static[F[_], A, B] = F[A => B].
