Monoid

A monoid is a semigroup with identity.

Typical imports

import scalaz.tc._
import scalaz.Scalaz._

Instance declaration

scala> {
     | import scalaz.Scalaz._
     | 
     | implicit def StringMonoid: Monoid[String] = instanceOf(new MonoidClass[String] {
     |   def mappend(a1: String, a2: => String) = a1 + a2
     |   val mempty = ""
     | })
     | }

Usage

scala> val s1 = "Hello"
s1: String = Hello

scala> val s2 = " World"
s2: String = " World"

scala> val s = s1.mappend(s2)
s: String = Hello World

scala> s.mappend(Monoid[String].mempty)
res1: String = Hello World

Law

A monoid instance must satisfy laws that make mempty into an identity for mappend, in addition to the Semigroup law:

scala> def leftIdentity[A, T](in: A)(assert: (A, A) => T)(implicit A: Monoid[A]): T =
     |   assert(in, A.mappend(A.mempty, in))
leftIdentity: [A, T](in: A)(assert: (A, A) => T)(implicit A: scalaz.tc.Monoid[A])T

scala> def rightIdentity[A, T](in: A)(assert: (A, A) => T)(implicit A: Monoid[A]): T =
     |   assert(in, A.mappend(in, A.mempty))
rightIdentity: [A, T](in: A)(assert: (A, A) => T)(implicit A: scalaz.tc.Monoid[A])T
