Semigroup

A semigroup is an algebraic structure consisting of a set and an associative binary operation.

A semigroup instance needs to satisfy the following law:

  • Associativity: append(x, append(y, z)) === append(append(x, y), z)

Typical imports

import scalaz.tc._
import scalaz.Scalaz._

Instance declaration

scala> implicit val intSemigroup: Semigroup[Int] = instanceOf(new SemigroupClass[Int] {
     |   def mappend(a1: Int, a2: => Int) = a1 + a2
     | })
intSemigroup: scalaz.tc.Semigroup[Int] = $anon$1@5c195226

Usage

scala> Semigroup[Int].mappend(1, 2)
res0: Int = 3

scala> 1.mappend(2)
res1: Int = 3
