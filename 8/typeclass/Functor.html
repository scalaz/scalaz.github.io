Functor

The Functor type class represents a type that can be mapped over.

A functor needs to satisfy the two functor laws:

  • Functor.map(elem)(identity) === elem
  • Functor.map(elem)(f.andThen(g)) === Functor.map(Functor.map(elem)(f))(g)

Typical imports

import scalaz.tc._
import scalaz.Predef._
import scalaz.Scalaz._

Instance declaration

scala> val listFunctor: Functor[List] = instanceOf(new FunctorClass[List] {
     |   def map[A, B](ma: List[A])(f: A => B): List[B] = ma match {
     |     case Nil => Nil
     |     case x :: xs => f(x) :: map(xs)(f)
     |   }
     | })
listFunctor: scalaz.tc.Functor[List] = $anon$1@1a95840f

Usage

Maybe does not have a map method. Instead its Functor is used, which is implicitly available from the above imports.

scala> val x = just(50L)
x: scalaz.data.Maybe.Maybe[Long] = Some(50)

scala> x.map(_ - 8L)
res0: scalaz.data.Maybe.Maybe[Long] = Some(42)
