Debug

Debug presents a safe and explicit alternative to Object#toString. By providing instances of this typeclass, a type explicitly defines that it can be converted into human-readable output.

Typical imports

import scalaz.Scalaz._

Built-in instances

scala> "Scalaz 8".debug
<console>:16: error: value debug is not a member of String
       "Scalaz 8".debug
                  ^

Instance declaration

scala> import scalaz.debug.DebugClass
<console>:15: error: object debug is not a member of package scalaz
       import scalaz.debug.DebugClass
                     ^
scala> import scalaz.data.Cord
<console>:15: error: object data is not a member of package scalaz
       import scalaz.data.Cord
                     ^
scala> case class Foo(a: Int)
defined class Foo

scala> implicit val fooDebug: Debug[Foo] =
     |   DebugClass.instance[Foo](a => Cord.wrap("Foo[", Cord(a.toString), "]"))
<console>:17: error: not found: type Debug
       implicit val fooDebug: Debug[Foo] =
                              ^
<console>:18: error: not found: value DebugClass
         DebugClass.instance[Foo](a => Cord.wrap("Foo[", Cord(a.toString), "]"))
         ^
<console>:18: error: not found: value Cord
         DebugClass.instance[Foo](a => Cord.wrap("Foo[", Cord(a.toString), "]"))
                                       ^
<console>:18: error: not found: value Cord
         DebugClass.instance[Foo](a => Cord.wrap("Foo[", Cord(a.toString), "]"))
                                                         ^

Usage

     | val bad = Foo(1).toString
<console>:3: error: illegal start of simple expression
       val bad = Foo(1).toString
       ^
     | val good = Foo(1).debug
<console>:3: error: illegal start of simple expression
       val good = Foo(1).debug
       ^
     | 
     | import scalaz.data.Maybe
<console>:4: error: illegal start of simple expression
       import scalaz.data.Maybe
       ^
     | Maybe.just(Foo(1)).debug
<console>:17: error: not found: type Debug
       implicit val fooDebug: Debug[Foo] =
                              ^
<console>:19: error: not found: value Maybe
       Maybe.just(Foo(1)).debug
       ^

Note that the Debug instance for Maybe resolves against the Debug instance for Foo.

String interpolator

Scalaz contains a custom string interpolator that takes advantage of Debug. Inside the z String interpolation, only objects that have Debug instances can be used.

     | case class Bar(a: Int)
<console>:4: error: illegal start of simple expression
       case class Bar(a: Int)
       ^
     | 
     | val foo = Foo(123)
<console>:5: error: illegal start of simple expression
       val foo = Foo(123)
       ^
     | val bar = Bar(234)
<console>:5: error: illegal start of simple expression
       val bar = Bar(234)
       ^
     | 
     | val interpolated = z"${foo}"
<console>:6: error: illegal start of simple expression
       val interpolated = z"${foo}"
       ^

The following fails to compile as there is no Debug instance for Bar

     | z"${bar}"
<console>:17: error: not found: type Debug
       implicit val fooDebug: Debug[Foo] =
                              ^
<console>:21: error: not found: value bar
       z"${bar}"
           ^
