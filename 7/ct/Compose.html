Compose

Compose[F, G, X] is a wrapper around the composition of type constructors F and G, applied to X. It is isomorphic to F[G[X]], but comes with instances derived by combining instances for F and G.

The following instances are provided:

if F is… and G is… then Compose[F, G, ?] is …
a Functor a Functor a Functor
an Apply an Apply an Apply
an Applicative an Applicative an Applicative
an InvariantFunctor an InvariantFunctor an InvariantFunctor
Contravariant a Functor Contravariant
a Functor Contravariant Contravariant
Foldable Foldable Foldable
